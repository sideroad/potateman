# How to play

1. Access http://potateman.herokuapp.com
2. Read QR code by players
3. Click Crash button to start game

* You can share mirroring URL of your stage by clicking share icon.

# Controller

| Command | Action|
|---------|-------|
|← or → | Move |
|↑| Jump |
|↓| Squat |
|A| Punch ( sinkable ) |
|↓ + A | Punch ( sinkable ) |
|↑ + A | Punch ( sinkable ) |
|B| Gard |
|B ( when flying ) | Floating |
|← or → + A | Dash |
|↓ + B | Ignite down speed |
|C| Meteorite |
|↓ + C | Thunderbolt |
|↑ + C | Volcano |
